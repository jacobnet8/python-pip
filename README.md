# Game Project

Para correr el juego debes seguir las siguientes
instrucciones en la terminal:

```sh
cd game
python3 main.py
```



# App Project

```sh
git clone
cd app
python3 -m venv env-app # Aqui podras crear tu entorno ambiente #
source env-app/bin/activate
pip3 install -r requirements.txt
python3 main.py
```
